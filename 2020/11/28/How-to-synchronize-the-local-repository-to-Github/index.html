How to synchronize the local repository to Github

2020-11-28
#github
Project Mangagement

The basic premise:

  • We have a github account.
  • We have already install ‘git’ in ubuntu system.
  • The local git obtain the submission permission from github.
  • We have a project fold in our local system. We assume that the path of the fold is ‘/home/user/my_project’

Let’s do it

‘’’
$ cd /home/user/my_project
$ git init ## this line will a fold ‘.git’ in your repository, which you can’t see it directly.
$ git add . ## Add all project documents to the staging area
$ git commit -m ‘Write your Comment here’ ## the files will be submit to the repository
‘’’

Build a new repository in Github for your project

Copy the address of your repository in Github. It looks like ‘https://github.com/yourName/yourRepo.git'
And then
‘’’
$ git remote add origin https://github.com/yourName/yourRepo.git
$ git push origin master
‘’’
